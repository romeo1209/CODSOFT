# CODSOFT Python Programming Internship

This repository contains the projects completed during the **CODSOFT Python Programming Internship**.

The purpose of these projects is to practice Python programming concepts including:

- GUI development
- Application logic
- Image handling
- Animations
- User interaction

---

# Projects Included

## 1. Calculator

A graphical calculator application built using **Tkinter** that supports basic arithmetic operations and some additional functions such as square and square root.

Features:

- Basic arithmetic operations
- Square and square root
- Percentage calculation
- Calculation history
- Light and dark themes

Technology Used:

- Python
- Tkinter
- Math module

---

## 2. Floating Contact Book

A visually styled **contact management system** built with **PyQt6**.

Features:

- Add contacts
- Display contact list
- Animated page transitions
- Custom GUI design
- Image-based interface

Technology Used:

- Python
- PyQt6

---

## 3. Rock Paper Scissors Game

A graphical version of the classic **Rock Paper Scissors** game where the player competes against the computer.

Features:

- Player vs computer gameplay
- Image-based interface
- Score tracking
- Multiple rounds

Technology Used:

- Python
- Tkinter
- Pillow
- Random module
